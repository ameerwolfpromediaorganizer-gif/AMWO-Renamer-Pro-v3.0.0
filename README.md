# 🐺 AMWO Renamer Pro+ v3.0.0

> Smart renaming tool for TV series & subtitle files — Press & Done

## ✨ Features
- Recognizes any naming pattern no matter how complex
- Deep-scans all nested subfolders
- Removes leading zeros automatically (S001E005 → S01E05)
- Preserves subtitle language codes (.ar .en .fr ...)
- Cleans Arabic & symbols from movie folder names
- Displays poster & artwork while processing
