## ✨ Features (المميزات)
- **Pattern Recognition:** Recognizes any naming pattern no matter how complex.
- **Deep Scan:** Scans all nested subfolders for a complete library cleanup.
- **Smart Formatting:** Automatically removes leading zeros (S001E005 → S01E05).
- **Subtitle Support:** Preserves language codes (.ar .en .fr ...) during renaming.
- **Clean Naming:** Cleans Arabic text and symbols from movie/series folders.
- **Visual Interface:** Displays poster & artwork while processing files.
